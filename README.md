### How to run it
You need node.js version 12 or higher.
First setup MySql data base, you can find .sql file in project root directory.
When you have working MySql do these steps:
```sh
$ cd game-swap
$ npm install
$ cd client
$ npm install
$ cd ../
$ npm run dev
```
